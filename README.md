Вэлком! Добро пожаловать на GitHub проекта Renaissance, где мы занимаемся презервацией протокола мессенджера Агент@Mail.ru. Наш проект появился в 2024 году и с тех пор активно развивается небольшой командой разработчиков.

Наша команда старается восстановить работу «Агента» в полном объёме, в том числе используя реверс-инженеринг и описания протокола других пользователей сети.

## Ресурсы

* [Документация к протоколу](https://github.com/tutmonde/mrim-docs)
* [Сервер-эмулятор](https://github.com/mrimsu/mrim-server)
* [Патч для клиентов для Windows](https://github.com/mrimsu/RenaissancePatcher)
